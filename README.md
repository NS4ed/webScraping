# webScraping

Uses node.js and JavaScript to build JSON object from website's html

Demo Project Collects the following from https://www.bls.gov/bls/ofolist.htm:
•	State
•	Director
•	Address
•	Phone
•	Fax
•	Email
•	URL

Save the JSON file (LMI.json) to the filesystem.

Libraries:
 - Cheerio
 - node-fetch
 - cors
